IFRAME (according to Mr. Calvario)

An iframe within html is used to display a web page within another webpage use cases may include bringing in over a form, videos, a map, and audo files.
  • because it is an inline element you can set a height and width too!
  • src defines the url of what should be embeded.
  • the iframe is an inline element.
  • you can remove the border by using border:none;
